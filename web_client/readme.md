# Python API Client Retry Demo using `requests`

To use, run the following commands from the `web_client` directory:

Windows: 

```bash
$ python -m venv .venv
$ ./.venv/Scripts/activate
$ pip install -r requirements.txt
$ python ./app/web_client.py
```

Linux: 

```bash
$ python -m venv .venv
$ ./.venv/bin/activate
$ pip install -r requirements.txt
$ python ./app/web_client.py
```
